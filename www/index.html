Jinx

Welcome to Jinx! Jinx is a versatile server software that offers flexible web infrastructure solutions. It supports various features like HTTP, HTTPS, forward proxy, reverse proxy, load balancing, and SSL termination. You can easily configure these features using a JSON file.
